What is JDBC?
    UI Testing: Java + Selenium Code - Driver Manager - Browser
    DataBase Testing : JAVA code - JDBC API - Driver(dedicated to that certain RDBMS) - Databases

    * Connection String : URL + UserName + Password
    * Driver of the Database : defined in the pom.xml as dependency

JDBC helps us with code re-usability

How we are using JDBC?
    1. Connect to DB
    2. Execute some queries
    3. Retrieve your result and manipulate the data that comes
